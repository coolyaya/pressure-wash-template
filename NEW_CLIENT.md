# New Client Setup Checklist

## 1) Clone + rename
- Copy repo to new folder/repo
- Update README/name as needed

## 2) Update site.json
Update:
- businessName
- phone
- email (public)
- city
- serviceAreas
- services
- seo defaults
- theme.brand colors
- theme.radius
- assets.logoText

## 3) Replace images
Replace:
- public/hero.jpg
- public/og.jpg
- public/favicon.ico

## 4) Vercel env vars
Set:
- RESEND_API_KEY
- LEAD_TO_EMAIL

## 5) Deploy + domain
- Deploy
- Connect domain
- Verify SSL

## 6) Google Search Console
- Verify domain
- Submit sitemap
